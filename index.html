Projects

Environmental Monitoring System

This project is an IoT-based Environmental Monitoring System developed using the ESP32 microcontroller in combination with sensors like DHT11, MQ135, and LDR. The system is capable of accurately detecting and displaying key environmental parameters such as Air Quality Index (AQI), temperature, humidity, and ambient light levels. The DHT sensor captures temperature and humidity, the MQ135 sensor monitors air quality, while the LDR tracks changes in light intensity. This project serves as a compact and efficient solution for real-time environmental data monitoring, with potential applications in smart homes, greenhouses, and urban pollution tracking.
